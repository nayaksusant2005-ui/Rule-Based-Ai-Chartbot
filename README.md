# 🤖 Rule-Based AI Chatbot

A simple Rule-Based AI Chatbot developed using Python as part of my Artificial Intelligence Internship at DecodeLabs.

## Features

- Responds to greetings and common questions
- Uses dictionary-based response mapping
- Handles user input efficiently
- Continuous chat loop
- Exit commands (`bye`, `exit`, `quit`)
- Fallback response for unknown inputs

## Technologies Used

- Python
- Dictionaries
- Loops
- Conditional Statements

## How to Run

```bash
python chartbot.py
```

## Example

```text
You: hello
Bot: Hi there! 👋

You: python
Bot: Python is a popular programming language.

You: bye
Bot: Goodbye! Have a great day! 👋
```
